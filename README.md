# Project Krypton
Krypton is a 2D online multiplayer battle arena built only with Java's built-in libraries.
## Demo
A video demo by my teammate Mark Zhang can be found on [YouTube](https://www.youtube.com/watch?v=_EjB9Uj-_Ww).
